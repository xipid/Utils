# Introduction

Welcome to the official documentation for the **Utils** repository. This collection consists of high-performance shell utilities designed to streamline Linux system administration, security management, and development workflows.

## Philosophy

The utilities in this repository are built with the following principles:

1.  **Automation First**: Eliminate repetitive manual tasks through intelligent scripts.
2.  **Zero Configuration**: Provide sensible defaults that work out-of-the-box while remaining highly customizable.
3.  **Portability**: Minimize external dependencies and support major Linux distributions (Fedora, Debian/Ubuntu, Arch).
4.  **Security**: Ensure secrets and sensitive data are handled with industry-standard practices.

## Overview of Components

- **Secrets Management**: `ke.sh` integrates KeePassXC with system authentication.
- **Data Synchronization**: `sy.sh` provides robust Rclone-based backups with advanced filtering.
- **Package Management**: `ipkg.sh` and `ghr.sh` simplify the installation of software from diverse sources.
- **Emulation Wrappers**: `wrap.sh` facilitates cross-architecture execution via QEMU.

Each utility is documented in detail within its respective section.
